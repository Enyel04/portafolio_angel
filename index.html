Autogestión Snitch

Desarrollado con: Flutter & Dart (Integrado con Idempiere ERP)

Plataforma B2B de autogestión orientada a optimizar la cadena de suministro de negocios. Simplifica el proceso de abastecimiento permitiendo a los comerciantes explorar catálogos, centralizar compras y generar pedidos de reposición de inventario en tiempo real al conectarse directamente mediante API al ERP corporativo.

Fuerza de Ventas

Desarrollado con: Flutter & Dart (Integrado con Idempiere ERP)

Aplicación móvil corporativa diseñada para optimizar las operaciones de equipos comerciales en campo. Permite la gestión eficiente de órdenes de venta, registro de cobros y captura de coordenadas geográficas.
